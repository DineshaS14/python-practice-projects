Rock Paper Scissors Game
A simple implementation of the Rock Paper Scissors game using Python and the Tkinter library.
Allows users to play against the computer.
Uses a random choice function to generate the computer's move.
Tic Tac Toe Game
A simple implementation of the Tic Tac Toe game using Python.
Allows two players to play against each other on a 3x3 grid.
Uses a command-line interface to input moves.
Weather API Fetcher
A script that fetches weather data using the Zipcode API and (link unavailable) API.
Takes a zip code as input and fetches the current temperature and image URL.
Uses the requests library to make API calls.
